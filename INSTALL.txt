 To install terse.py:
   1. Download and unpack Jmol in the webroot folder
   2. Download and unpack terse.py (https://github.com/talipovm/terse/archive/master.zip) one of the $PATH folder, e.g. ~/bin
   3. Copy configuration file to the home folder ~/.terse.rc
   4. Copy terse.css file to the webroot folder
   5. Modify the configuration file ~/.terse.rc (The most important options are OutputFolder and JmolPath)
   6. Install gnuplot (if not already installed) and add it's location to $PATH
